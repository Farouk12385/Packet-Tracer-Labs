# Packet Tracer Labs

## Overview
Welcome to the Packet Tracer Labs repository! This repository contains various labs that help in understanding and implementing networking concepts using Cisco's Packet Tracer.

## Directory Structure
```
Packet-Tracer-Labs/
├── Lab1_Basics/
│   ├── Lab1_Description.md
│   ├── Lab1_File.pkt
│   └── Lab1_Resources/
├── Lab2_VLANs/
│   ├── Lab2_Description.md
│   ├── Lab2_File.pkt
│   └── Lab2_Resources/
├── Lab3_ROUTING/
│   ├── Lab3_Description.md
│   ├── Lab3_File.pkt
│   └── Lab3_Resources/
└── Lab4_SECURITY/
    ├── Lab4_Description.md
    ├── Lab4_File.pkt
    └── Lab4_Resources/
```

## Installation and Usage
1. Clone the repository: `git clone https://github.com/Farouk12385/Packet-Tracer-Labs.git`
2. Open Packet Tracer and load the relevant lab files.
